# Wroclaw Sales Analysis

## Project Objective
This project analyzes sales performance for a retail chain in the Wroclaw region to optimize inventory and identify top-selling product categories.

## Data Structure
- `Raw_Data/`: Contains the initial transaction logs (`sales_data.csv`).
- `SQL_Queries/`: Contains SQL scripts used to extract revenue and performance metrics.

## Technical Skills Demonstrated
- Data Cleaning with Python/Pandas.
- Database querying using SQL (Joins, Aggregations).
- Sales trend analysis and reporting.

## Key Insights
- Identified top-performing product categories.
- Optimized revenue tracking per unit sold.
